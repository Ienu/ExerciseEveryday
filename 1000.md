## 1000: A+B Problem

总时间限制: 1000ms    内存限制: 65536kB

### 描述

Calculate a + b

### 输入

Two integer a, b (0 ≤ a, b ≤ 10)

### 输出

Output a + b

### 样例输入
```
1 2
```
### 样例输出
```
3
```
这个题就是个入门的题目，让大家熟悉一下输入输出的格式。我最擅长的语言是C++，不过可能有的时候也会带点C，先贴代码吧
```cpp
#include <iostream>
using namespace std;
int main()
{
	int a, b;
	cin >> a >> b;
	cout << a + b << endl;
	return 0;
}
```
