## 4139: 不定方程求解

总时间限制: 1000ms    内存限制: 65536kB

### 描述

给定正整数a，b，c。求不定方程 ax+by=c 关于未知数x和y的所有非负整数解组数。

### 输入

一行，包含三个正整数a，b，c，两个整数之间用单个空格隔开。每个数均不大于1000。

### 输出

一个整数，即不定方程的非负整数解组数。

### 样例输入
```
2 3 18
```
### 样例输出
```
4
```
由于解均为非负整数，所以在穷举x,y的时候从0开始，到c/a，和c/b，即x最大时，y为0，反之亦然。
```cpp
#include <iostream>
using namespace std;
int main() {
	int a, b, c, i, j, sum = 0;
	cin >> a >> b >> c;
	for (i = 0; i <= c / a + 1; ++i) {
		for (j = 0; j <= c / b + 1; ++j) {
			if (a * i + b * j == c) {
				sum++;
			}
		}
	}
	cout << sum << endl;
	return 0;
}
```
