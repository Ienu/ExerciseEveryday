## 2854: 点和正方形的关系

总时间限制: 1000ms    内存限制: 65536kB

### 描述

有一个正方形，四个角的坐标（x,y)分别是（1，-1），（1，1），（-1，-1），（-1，1），x是横轴，y是纵轴。写一个程序，判断一个给定的点是否在这个正方形内。

### 输入

输入坐标x，y

### 输出

yes或者no

### 样例输入
```
3 4
1 1
0 0
```
### 样例输出
```
no
yes
yes
```
从给定的样例来看，正方形边缘上的点也计算在正方形内。无限输入的要求可以通过while循环来实现。
```cpp
#include <iostream>
using namespace std;
int main() {
	double x, y;
	while (cin >> x >> y) {
		cout << ((x >= -1 && x <= 1 && y >= -1 && y <= 1) ? "yes" : "no") << endl;
	}
	return 0;
}
```
