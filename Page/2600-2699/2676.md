[题目来源](http://bailian.openjudge.cn/practice/2676/)

## 2676: 整数的个数

总时间限制: 1000ms    内存限制: 65536kB

### 描述

给定k（1 < k < 100）个正整数，其中每个数都是大于等于1，小于等于10的数。写程序计算给定的k个正整数中，1，5和10出现的次数。

### 输入

输入有两行：第一行包含一个正整数k，第二行包含k个正整数，每两个正整数用一个空格分开。

### 输出

输出有三行，第一行为1出现的次数，第二行为5出现的次数，第三行为10出现的次数。

### 样例输入
```
5
1 5 8 10 5
```
### 样例输出
```
1
2
1
```
因为所有的数都是1到10之间的数，所以可以使用箱子对全部的值直接计数，最后输出指定的值的个数即可。
```cpp
#include <iostream>
using namespace std;
int main() {
	int k, i, temp;
	cin >> k;
	int count[10] = { 0 };
	for(i = 0; i < k; ++i) {
		cin >> temp;
		count[temp - 1]++;
	}
	cout << count[0] << endl
		<< count[4] << endl
		<< count[9] << endl;
	return 0;
}
```
