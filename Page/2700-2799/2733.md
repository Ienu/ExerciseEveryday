## 2733: 判断闰年

总时间限制: 1000ms    内存限制: 65536kB

### 描述

判断某年是否是闰年。

### 输入

输入只有一行，包含一个整数a(0 < a < 3000)。

### 输出

一行，如果公元a年是闰年输出Y，否则输出N。

### 样例输入
```
2006
```
### 样例输出
```
N
```
能被3200整除的不是闰年，其他的能被400整除的是闰年，其他的能被100整除的不是闰年，其他的能被4整除的是闰年。
```cpp
#include <iostream>
using namespace std;
int main() {
	int a;
	cin >> a;
	bool r = false;
	if((a % 4) == 0) {
		if((a % 100) != 0) {
			r = true;
		}
		else {
			if((a % 400) == 0 && (a % 3200) != 0) {
				r = true;
			}
		}
	}
	cout << (r ? "Y" : "N") << endl;
	return 0;
}
```
